Projetos

Economiza Já

Economiza Já é um sistema completo para o gerenciamento de finanças pessoais, desenvolvido com foco na organização e controle financeiro de forma simples e intuitiva.

Back-End

Construído utilizando Java Spring Boot e o banco de dados PostgreSQL, garantindo segurança e eficiência no armazenamento de dados.

Front-End Web

Desenvolvido com Angular e TypeScript, oferece uma interface amigável e dinâmica, permitindo que os usuários acompanhem suas finanças diretamente do navegador.

Front-End Mobile

Implementado em Flutter com Dart, oferecendo uma experiência consistente e responsiva em smartphones.
